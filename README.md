# Hotel Reservation System

A C++ console application for managing hotel room reservations using a Binary Search Tree (BST).

## Features
- Add rooms
- Book a room
- Cancel booking
- Search for a room
- Display available rooms
- Display booked rooms

## Technologies
- C++
- Object-Oriented Programming (OOP)
- Binary Search Tree (BST)
- Visual Studio
